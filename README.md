# 🚀 Максимально быстрая проверка Vless/SS/Trojan серверов + флаги стран (эмодзи) — «КРОТовые ТОННЕЛИ»

**Автоматический сбор, проверка и формирование подписки с рабочими VLESS-серверами.**  
Скрипт парсит ссылки из списка источников, тестирует их доступность (TCP + реальный прокси-тест через Xray-core) и генерирует готовую подписку с красивыми тегами, флагами стран и датой обновления.

---

## ✨ Возможности

- 📥 **Сбор ссылок** из текстовых источников (прямые `vless://` ссылки, Base64‑подписки, обычные файлы) — поддерживается файл `sources.txt`.
- ⚡ **Двухступенчатая проверка**:
  1. Быстрый TCP‑коннект (массово, до 400 параллельных проверок).
  2. Полноценный тест через локально поднятый **Xray-core** (с поддержкой TLS, WS, Reality).
- 🌍 **Автоматическое определение страны** и добавление эмодзи-флага 🇷🇺🇺🇸🇩🇪 (требуется база GeoLite2).
- 🏷 **Умная нумерация серверов** вида `#СЕРВЕР 0001 | 🇷🇺 | ОБНОВЛЕН 05-03-2026`.
- ⏱ **Отсев медленных серверов** (порог настраивается).
- 📄 **Генерация подписки** в двух форматах:
  - `subscription.txt` — plain text с заголовками и тегами.
  - `subscription_base64.txt` — Base64‑версия для использования в клиентах.
- 🔧 Гибкая настройка (количество потоков, таймауты, порты) прямо в скрипте.

---

## 📋 Требования

- **Python 3.7+**
- **Xray-core** (установлен и доступен в PATH или указан путь в переменной `XRAY_CORE_PATH`)
- Опционально: **geoip2** и база **GeoLite2-Country.mmdb** (для флагов)
